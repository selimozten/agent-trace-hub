# Support

Use [GitHub Discussions](https://github.com/selimozten/agent-trace-hub/discussions)
for setup questions, adapter ideas, and training-format discussions.

Use [GitHub Issues](https://github.com/selimozten/agent-trace-hub/issues) for
reproducible bugs and scoped feature requests. Include `ath --version`, your
operating system, the source adapter, and a synthetic or fully redacted sample.

Report security issues privately as described in [SECURITY.md](SECURITY.md).
